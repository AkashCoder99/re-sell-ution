# ReSellution

## Project Description
ReSellution is a full-stack local marketplace platform that enables users to list, discover, and purchase pre-owned items within their community. Users can create listings with images and details, browse and search items by category and location, manage their posts, and communicate securely with interested buyers/sellers. The project focuses on building a scalable, user-friendly system using modern software engineering practices such as modular design, authentication, and RESTful APIs.
